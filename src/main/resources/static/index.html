소셜 로그인 테스트

브라우저에서 각 소셜 로그인 인가 화면으로 이동한 뒤, callback URL이 백엔드로 정상 복귀하는지 확인하는 테스트 페이지입니다.

현재 callback URL 기준
  • http://localhost:8080/login/oauth2/code/kakao
  • http://localhost:8080/login/oauth2/code/google
  • http://localhost:8080/login/oauth2/code/naver
  • 성공하면 provider 로그인 후 백엔드 callback endpoint가 JSON 응답을 반환합니다.
  • 네이버는 state 값이 필요해서 테스트 페이지에서 자동 생성합니다.
  • 구글/카카오/네이버 콘솔에 동일한 callback URL이 등록되어 있어야 합니다.
